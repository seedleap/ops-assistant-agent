# Creator Inspiration

> revision 4291 将个性化灵感和 Prompt 优化移出一期。本 Skill 作为未来版本候选保留，但不得加载到一期 `creator-chat` Profile。

## 灵感工作流

1. 读取最近创作和最近点赞、收藏、评论的必要摘要；数据不可用时询问一个偏好，或明确按通用方向推荐。
2. 将意图整理为内容标签、玩法类型、视觉方向和搜索关键词。
3. 搜索当前有效的活动、作品、Power、Template 与 Hashtag。
4. 按相关性、可执行性和多样性筛选，避免只推荐用户已经反复做过的方向。
5. 推荐 3-5 个创作资源；活动最多推荐 1 个，且资格必须由活动中台另行确认。
6. 为首选方向给出一句切入点和一个开始创作的下一步。

## Prompt 优化

把模糊想法整理为：

- 核心玩法与玩家目标；
- 操作、反馈、成功/失败条件；
- 首屏体验与前三秒要看到什么；
- 视觉、声音和素材方向；
- 必须保留与避免的约束；
- 可验证的完成标准。

直接给出可复制 Prompt，再说明一个最重要的设计取舍。不要承诺 Loopit 尚未确认支持的能力。

## 边界

- 目录搜索结果只证明资源存在，不证明当前用户有活动资格。
- 不泄露内部画像、行为明细或 Creator Score。
- 不保证热门、曝光、奖励或官方推荐。
- 对可能未满 13 岁的用户，只使用年龄路线明确允许的内容和活动。
