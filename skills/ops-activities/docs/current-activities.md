# 当前运营活动

> 运营同学维护：在这里增改活动。每个活动写清「链接 + 介绍 + 适合谁 + 怎么参加」，创作小助手会据此给创作者推荐。

## 活动一 · 社区精选合集（第一期）

- **链接**：https://share.loopit.me/game/66a911fd-c946-43b3-b7f2-b2b58bc9b0f6
- **形式**：官方精选合集，会挑选一批社区里表现亮眼的作品，集中放进合集里给大量曝光。
- **适合谁**：作品有亮点、数据或口碑不错、想要更多曝光的创作者，争取被选入合集。
- **怎么参加**：把作品做出记忆点（前 3 秒抓人、玩法清晰、画风统一），保持活跃；运营会从热门作品里挑选入选。
- **氛围**：玩家互动很热（大量点赞评论，很多人留言"求把我的游戏加进来"），入选能带来明显曝光。

## 活动二 · 社区精选合集（第二期）

- **链接**：https://share.loopit.me/game/c1143e02-0a49-4793-80e0-d8274c9e989f
- **形式**：同上，新一期的官方精选合集，轮换展示一批优质社区作品。
- **适合谁**：还没进过合集、近期有新作或数据在涨的创作者，是争取曝光的好机会。
- **怎么参加**：持续产出、打磨作品质量，提升点赞/评论/完播等表现，增加被选中的概率。
