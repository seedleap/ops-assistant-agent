# 热榜 case 解析：值得学的爆款做法

挑了几个近期真实上热门、适合当模板学习的作品，拆给创作者看"它们为什么火、是怎么做的、你能学什么"。

## Case 1 · Spin the Bottle —— 极简一句话也能爆

- **去玩**：https://share.loopit.me/game/6c3cf1db-1c76-4ed4-819c-10d019a23463
- **数据**：约 164 万曝光、1.7 万赞、3262 条评论、10s 播放率 ~75%（互动量爆表）。
- **当初的 prompt**（就一句话）：`pick 4 characters spin the wheel whoever the bottle lands on kiss them`（选4个角色转盘，瓶子指到谁就亲谁）。
- **它怎么做的**：4 个动漫角色围一圈 + 经典玻璃瓶 + 派对背景，点 SPIN 转瓶子看结果。
- **能学什么**：
  - **玩法自带社交话题**："转瓶子/亲谁"这种带点暧昧和悬念的设定，天然让人想玩、想截图、想喊朋友——评论区一片"Yayyy"就是情绪被点着了。
  - **一键即玩**：进来就一个按钮，零门槛、可重复。
  - **角色亲和力**：动漫角色比抽象图形更容易让人有代入感。
  - 👉 **不是 prompt 越长越好**，关键是**玩法点子本身有没有传播力**。

## Case 2 · Human Shape Puzzle —— 详细设计稿换来高完成度

- **去玩**：https://share.loopit.me/game/9def95b6-6a23-406c-a3df-7330ab07eaed
- **数据**：约 86 万曝光、1.3 万赞、2007 条评论、10s 播放率 ~76%、3D。
- **当初的 prompt**（详细设计稿）：做一个竖屏解谜「Human Shape Puzzle」，玩家控制一个可弯曲的人偶，**旋转关节去匹配目标剪影**，完全对上就过关，手感要"satisfying（解压、满足）"。
- **它怎么做的**：Low poly 3D + 柔和粉彩配色（React Three Fiber），把"调姿势对剪影"的爽感机制做扎实。
- **能学什么**：
  - **把一个"解压爽感"机制讲清楚**：目标明确（对上剪影）、即时反馈、完成有满足感——这类机制留存高（10s 率 76%）。
  - **画风统一精致**：低多边形 + 粉彩，观感干净一致，加分明显。
  - 👉 **想要高完成度/高完播**，就把**核心机制 + 画风**在 prompt 里描述清楚**，别只丢一句话。

## 两条都能走的成功路径

1. **极简 + 话题**：一句话的点子，但玩法自带社交/情绪钩子（像转瓶子、整蛊、暧昧、攀比）。门槛低、传播快。
2. **详细 + 爽感 + 统一画风**：把核心机制和美术风格写清楚，做出解压/满足感和精致观感。留存和口碑更稳。

## 热榜共性（看了一批 Top 作品）

- 几乎都是 **10s 播放率 ~75%** 起步——**前几秒抓住人**是硬门槛。
- 题材很杂（解谜/模拟/艺术/meme/运动都有）——**题材不设限，赢在玩法爽点 + 画风统一**。
- **评论量都很高**——玩法自带"想晒、想聊、想喊人来"的钩子，是上热门的助推器。
