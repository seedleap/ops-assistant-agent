# Loopit 能力介绍：你能做出什么

Loopit 是一个"**一句话就能做出可玩小游戏**"的 AI 创作平台。你用自然语言描述想法，Loopit 的创作 agent 会帮你把设计、美术、音乐、代码到打包全链路搞定，最快几分钟就能拿到一个能在手机上直接玩、直接分享的小游戏。

## 一、你只要会"描述"，剩下的交给它

- **一句话起步**：「做一个越吃越胖的放置游戏」「画只会跳的小鸡」「来个配色解谜」——给个概念就能生成可玩原型。
- **也能给详细稿**：你可以指定玩法细节、画风、主题、氛围、配色，它会照着做。
- **对话式打磨**：做出来不满意？继续聊着改——换美术、调难度、加玩法、修 bug，多轮迭代直到满意。

## 二、能做哪些类型的游戏

- **超休闲 / Hypercasual**：点一点、滑一滑就上瘾的循环玩法（最容易上手、最容易传播）。
- **放置 / 挂机（Idle）**：数值滚雪球、越玩越爽（爆款高发区）。
- **解谜 / Puzzle**：关卡、连连看、配色、拼图、逻辑。
- **养成 / 装扮（Dress-up）**：拖拽换装、捏角色、布置场景。
- **动作 / 街机**：躲避、跳跃、竞速、反应类。
- **交互体验**：不一定是"游戏"，也可以是好玩的互动动画、整活小工具、故事体验。

## 三、美术、音乐、音效——它直接帮你生成

不用自己画、自己找素材，Loopit 能按你的描述直接生成：

- **图片 / 角色 / 背景**（可指定画风：卡通、写实、3D、手绘、像素、极简、赛博朋克……）
- **逐帧动画精灵**（让角色动起来）
- **短视频片段**
- **背景音乐 + 音效 + 配音**（指定情绪/风格）
- **也能用你自己上传的图/音/视频**当素材或参考

> 想要统一好看的观感，描述时带上**画风 + 配色 + 氛围**（比如"扁平卡通、马卡龙配色、轻松治愈"），出来的效果会更整。

## 四、进阶玩法（让作品更有料）

- **Remix 二创**：基于已有作品改造——换皮、改数值、加模块，快速产出新版本。
- **智能玩法**：可以接入 AI——会聊天的 NPC、根据你输入实时生成图片、文字理解等。
- **设备能力**：摄像头、麦克风、陀螺仪、震动、拍照选图，做体感/AR 类玩法。
- **社交与排行**：排行榜、点赞/收藏/评论计数、存档读档、分享二维码——这些能让作品更容易被反复玩和传播。

## 五、它替你把好"质量关"

- 自动**预览**，确认好再发布。
- 内置**类型检查 + 构建校验**，做不出来会自动重试修复，尽量保证"能跑、不白屏"。
- 默认遵循好玩游戏的几条铁律（见下），帮你避开新手坑。

## 六、想更容易上热门，记住几条原则

- **前 3 秒要有"Wow"**：一进来就有视觉或玩法的爽点，留住人（10s 播放率是关键指标）。
- **竖屏单手可玩**：移动端标准尺寸（约 400×580），别做需要双手/横屏的复杂操作。
- **别一上来就放开始菜单**：默认直接进游戏，越快玩到越好。
- **聚焦一个爽点**：一个清晰、可重复、越玩越爽的核心循环，胜过堆一堆功能。
- **给社交留钩子**：排行榜、晒成绩、求挑战——让玩家有理由点赞、评论、分享。

## 七、目前还不太支持的

- 实时联机对战 / 多人同步（可以做排行榜式的"异步比拼"，但不是同屏 PVP）。
- 上传自训练的 AI 模型。
- 导出到 Unity/Godot 这类外部引擎（3D 走的是网页方案）。
