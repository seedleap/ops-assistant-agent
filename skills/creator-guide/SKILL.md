# 创作者指导

这是给**创作者**看的创作指导知识库。`docs/` 下每个 md 是一篇可独立编辑的指导文档，运营/产品同学可在前端"创作者指导"tab 里增删改。

创作小助手在聊到"怎么做爆款、有什么创作技巧、Loopit 能做出什么效果、有没有可以学的案例"时，会用 `read_knowledge('creator_guide')` 读取这里的内容来给建议。

默认包含：
- **Loopit 能力介绍**（`loopit-capabilities`）——从创作者视角讲 Loopit 能做出什么效果。
- **热榜 case 解析**（`hot-case-studies`）——值得学习的真实爆款案例与做法。
